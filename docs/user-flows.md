# User Flows

User Registration

User creates an account.  
User confirms account credentials.

Campaign Discovery

User opens the platform.  
User browses active campaigns.  
User selects a campaign.

Campaign Participation

User views campaign details.  
User selects product options.  
User proceeds to checkout.  
User completes payment.

Order Tracking

User accesses account dashboard.  
User views order history.  
User checks order status.
