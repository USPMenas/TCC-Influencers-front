# Mobile First Design

The platform must prioritize mobile usage.

Most users will access campaigns through social media links.

---

## Design Principles

Mobile layouts must be the primary layout.

Desktop layouts should adapt from mobile.

Touch interaction must be optimized.

Navigation should be simple and thumb-friendly.

---

## Performance Considerations

Images must be optimized.

Pages must load quickly on mobile networks.

Campaign pages must prioritize essential information.

---

## Campaign Page Priorities

Product image

Campaign progress

Purchase button

Campaign description

Sharing options
