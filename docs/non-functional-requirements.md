# Non Functional Requirements

Performance

Pages should load in less than 2 seconds under normal conditions.

Scalability

The system must support traffic spikes during campaign launches.

Availability

The platform should maintain high availability.

Security

All communication must use HTTPS.

User data must be stored securely.

Usability

The platform must be intuitive and easy to use.

Mobile Experience

Mobile experience must be prioritized over desktop experience.
