# Functional Requirements

RF01  
Users must be able to create an account.

RF02  
Users must be able to log in.

RF03  
Influencers must be able to create campaigns.

RF04  
Campaigns must include product information, price, goal, and deadline.

RF05  
Users must be able to browse active campaigns.

RF06  
Users must be able to support campaigns by purchasing products.

RF07  
The platform must track campaign progress.

RF08  
Users must be able to view their order history.

RF09  
Campaigns must automatically close when the deadline is reached.

RF10  
Campaigns must succeed when the goal is reached.

RF11  
Campaigns must fail when the goal is not reached.
